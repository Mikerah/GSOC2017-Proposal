Abstract

For GSoC, I plan to work on automating backlisting and adding more documentation to the Ahmia project's web pages. First, I will work my way through the codebase to learn how the search engine works. After this, I will add documentation to the project's github page by creating a wiki for new developers. Next, I will create support pages on the Ahmia search engine for advanced search features and for general help navigating the search engine. Next, I will start researching features of websites we want to blacklist such as content, malware,etc. My initial idea is to use a probabilistic approach. My other goals for the project include anonymously collecting data and adding a query language for advanced searching options.

Tasks

Why Tor?

Since my elementary school days, I have had a strong interest in all things technology. This includes security and artificial intelligence. In the 6th grade, I wanted to become a hacker. Thus, I read upon all the things hackers used and all the knowledge they have to gain in order to be successful hackers. During this period is how I discovered the Tor project and in particular the Tor browser. It was slow and I was ignorant about its significance. Since then, the Tor project has come such a long way and I have seen it become more and more on people's radar. Ever since Edward Snowden disclosed the NSA surveillance, I began using the Tor browser more regularly and developed a deeper interest in security. I realized that I was taking for granted how open and free the Internet is and why need organizations like the Tor project and EFF to defend what makes the Internet probably the most important human achievement in recent history.
