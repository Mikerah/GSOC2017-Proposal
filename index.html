Abstract

For GSoC, I plan to work on automating backlisting and adding more documentation to the Ahmia project's web pages. First, I will work my way through the codebase to learn how the search engine works. After this, I will add documentation to the project's github page by creating a wiki for new developers. Next, I will create support pages for the search engine to help the usage of Ahmia. Afterwords, I will develop an algorithm for blacklisting child abuse websites by search for sites that link to these types of websites. In order to make Ahmia more user friendly, I will be developing advanced search options, similar to the way Google's advanced search options work. My optional goals for the summer are to develop ways to anonymously collect user data, improving the current statistics page and ways to gather anonymous feedback directly from users.

The blacklisting algorithm will take advantage of the fact that linking to child abuse websites is a grey area, in other words, it is not illegal.The algorithm will acquire links to websites that link to child abuse websites, and ban the the links to the child abuse websites. Since search for particular websites due to the way Tor websites work, I think that having advanced search options will make it easier to browse websites on the Tor network. I propose the following search options:

  • The use of double quotes("") to search for a particular string and it is case sensitive. Example: "Politics" search term should only present websites that have a header/description with the word Politics
  • The use of the operator OR between search terms. Exmaple: term1 OR term2 should only present websites that contain either term1 or term2 or both term1 and term2 (like a logical OR)
  • The use of the oprator AND between search terms. Example: term1 AND term2 should only present websites that contain both term1 and term2.
  • The use of a dash(-) to before a search term. Example: -term1 should only present websites that don't contain term1.
As we get more user feedback, more search options will be added.
